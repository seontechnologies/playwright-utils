# Lessons Learned

```xml
<elicitation id="bmad/core/adv-elic/lessons-learned.md" name="Lessons Learned">
  <description>Advanced elicitation technique for lessons learned</description>

  <core-method>Extract specific, actionable insights from past experiences to inform current decisions and prevent future mistakes.</core-method>

  <llm>
    <action>Be specific - vague lessons don't prevent future mistakes</action>
    <action>Capture failures as they often teach more than successes</action>
    <action>Make lessons actionable with concrete steps</action>
  </llm>

  <final-report title="Lessons Learned Analysis">
    <value>CHALLENGE: [Current problem]</value>
    <value>PAST EXPERIENCES: [Similar situations]</value>
    <value>WORKED: [Successful approaches + why]</value>
    <value>FAILED: [Unsuccessful attempts + causes]</value>
    <value>INSIGHTS: [Key patterns extracted]</value>
    <value>APPLY: [How to use lessons now]</value>
  </final-report>
</elicitation>
```
