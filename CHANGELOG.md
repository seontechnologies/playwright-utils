# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.9.0] - 2025-11-19 (Current Release)

### Added

- Network recorder utility with HAR-based recording and playback
- Intelligent CRUD detection for stateful mocking
- Network error monitor for automatic HTTP error detection
- Burn-in utility with smart test filtering
- API request utility with schema validation support (JSON Schema, OpenAPI, Zod)
- Auth session management with token persistence
- File utilities for CSV, XLSX, PDF, ZIP handling
- Structured logging integrated with Playwright reports
- Network interception for request/response mocking
- Polling utility (recurse) for async conditions
- Dual module support (CommonJS and ES Modules)
- Comprehensive TypeScript type definitions
- Full Playwright fixtures integration
- "Functional core, fixture shell" pattern throughout

### Changed

- Open sourced under MIT license
- Repository made public at github.com/seontechnologies/playwright-utils
- Published to public npm registry for easier installation
